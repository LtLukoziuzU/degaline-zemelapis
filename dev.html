GitHub prieigos raktas

Norint išsaugoti patikrinimus tiesiai į repozitoriją iš naršyklės, reikia asmeninio GitHub rakto (PAT).

  1. Eikite į github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens
  2. Sukurkite naują raką su prieiga tik prie LtLukoziuzU/degaline-zemelapis
  3. Leidimai: Contents → Read and write
  4. Įklijuokite raką žemiau
